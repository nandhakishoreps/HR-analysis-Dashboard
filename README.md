# HR-analysis-Dashboard
PowerBI dashboard for HR analysis
An interactive Power BI dashboard designed to analyze employee attrition and workforce trends, enabling data-driven HR decisions.

This project focuses on transforming raw HR data into meaningful insights through effective data cleaning, modeling, and visualization. The dashboard presents key workforce metrics in a visually appealing and business-oriented format that helps stakeholders quickly evaluate attrition patterns and employee behavior.

Project Objective

The main objective of this dashboard is to:

Analyze employee attrition trends

Identify key factors contributing to attrition

Provide clear KPIs for workforce evaluation

Enable HR teams and management to make informed strategic decisions

Data Preparation & Cleaning

Before visualization, the dataset was carefully prepared using Power Query, including:

✔ Removal of unwanted and irrelevant columns
✔ Handling null values by replacing them appropriately
✔ Removing duplicate records
✔ Selecting only required columns for analysis
✔ Ensuring data consistency and accuracy

This ensures the dashboard is built on clean, reliable, and business-ready data.

📌 Key KPIs (Cards Visualization)

The dashboard uses Card visuals to highlight critical HR metrics at a glance:

👥 Total Number of Employees

🚪 Total Attrition Count

📉 Attrition Rate (%)

⭐ Average Performance Rating

💰 Average Salary

🕒 Average Years at Company

These KPIs provide an instant overview of the organization’s workforce health.

📊 Visualizations & Insights
🌳 Gender Comparison (Tree Map)

Tree Map used to compare Male vs Female employees

Helps identify workforce distribution and gender-based attrition patterns

🍩 Attrition by Education (Donut Chart)

Displays attrition across different education levels

Highlights which educational background shows higher attrition risk

📋 Job Satisfaction Analysis (Matrix)

Matrix visualization showing sum of attrition count by job satisfaction

Enables deeper analysis of how satisfaction levels impact attrition

📊 Attrition by Age Group (Bar Charts)

Multiple bar charts representing attrition across different age categories

Helps identify high-risk age groups

📈 Attrition Trend by Age (Line Chart)

Line chart used to show attrition trend across age

Makes it easy to observe patterns and shifts visually

💡 Business Value

This dashboard helps stakeholders to:

✔ Easily evaluate employee attrition trends
✔ Identify problem areas in workforce retention
✔ Understand demographic and satisfaction-based attrition
✔ Support HR strategy with visual evidence
✔ Make faster and more confident business decisions

The clean layout and interactive visuals ensure easy interpretation for both technical and non-technical users.
